# Hero Media

Drop homepage background images or videos in this folder.

Supported extensions:

- `.jpg`
- `.jpeg`
- `.png`
- `.webp`
- `.avif`
- `.mp4`
- `.webm`

Files are loaded automatically at dev/build time and sorted by filename, so names like `01-flight.mp4`, `02-map.png`, and `03-lab.jpg` control playback order.
